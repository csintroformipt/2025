Исследовать на устойчивость все положения равновесия в системе, описываемой уравнением движения:  
  F'' = - (a * nu**2 *cos(nu*t) + g) sinf(f)/l  
a, g, l - коэффициенты, nu - параметр для исследования устойчивости  
a - амплитуда гармонических колебаний точки подвеса физического маятника,  
g - сила тяжести,  
l - длина подвеса

Данное уравнение описывает движение маятника с движущейся по гармоническому закону точкой подвеса.
